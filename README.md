#### 상상하는 것을 현실로

사람들에게 멋진 가치를 제공 하는 것을 추구하는 개발자 이상현입니다.
실제 누구나 사용하는 프로덕트를 개발하여 사람들의 삶의 질을 높여 줄수 있는 개발자가 되고자 합니다.
최고의 프로덕트를 위해 최고의 코드를 고민하며, 경험을 통해 성장하며 발전하고자 합니다.
